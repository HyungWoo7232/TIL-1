# 실습1. TIL 폴더 git으로 활용하기

* TIL 폴더에 정리한 마크다운 파일을 넣고
  * tip! 우클릭 git bash here로 열기
* git 저장소로 설정
* commit!

```bash
$ git commit -m '...'
결과를 복사해서 채팅창에 던지기!
```
